O que é o IASD Bootstrap

O IASD Bootstrap é um framework de front-end elegante e intuitivo, desenvolvido para a tornar a criação de sites e portais da Igreja Adventista mais ágil e fácil. Foi construído com base no projeto Twitter Bootstrap.

Twitter Bootstrap

Twitter Bootstrap é uma coleção de ferramentas para criação de sites e aplicativos online gratuita. Contém templates para tipografia, formulários, botões, navegações e outros componentes de interfaces baseados em marcação HTML e estilos CSS, além de extensões opcionais em JavaScript.
O IASD Bootstrap é um fork da versão v3.1.1 do Twitter Bootstrap.

Clique aqui para ler a documentação oficial do Bootstrap

Download

Tecnologias e serviços utilizados

Sass

Sass é um acrônimo para Syntactically Awesome Style Sheets. É um pré-processador de CSS, que adiciona funcionalidades extras a este formato de arquivo, como a utilização de variáveis, expressões matemáticas, identação, herança de estilo, minificação entre outros.

Clique aqui para acessar o site do projeto

Compass

Compass é um framework que reúne diversas ferramentas para o desenvolvimento de folhas de estilo em Sass. Possui várias extensões ao Sass além de um próprio compilador de código, sprites e imagens. Ele roda em qualquer sistema que tenha Ruby instalado.

Clique aqui para acessar o site do projeto

Wordpress

Wordpress é uma ferramenta para a criação de blogs e também um sistema de gerenciamento de conteúdo (CMS - Contant Management System), baseado na linguagem PHP e banco de dados MySQL.
O Portal Adventista utiliza a versão v3.8.1.

Clique aqui para acessar o site do projeto

Google Fonts

Google Fonts é um diretório de fontes open-source para a web que podem ser utilizados para o desenvolvimento de sites através da Google Font API. O Portal Adventista utiliza a fonte "Lato" nos pesos Thin (100), Light (300), Normal (400), Bold (700) e Ultra-Bold (900), além das versões em itálico.

Clique aqui para acessar o site do projeto

Bibliotecas e plugins de terceiros

Jquery

jQuery é uma biblioteca JavaScript rápida, leve e rica em recursos. Ele simplifica o desenvolvimento de interações através de métodos simples, exigindo menos linhas de código para criação de requisições em AJAX ou a manipulação de elementos no DOM.
O Portal Adventista utiliza a versão v1.10.2.

Clique aqui para fazer o download da biblioteca

FontAwesome

FontAwesome é uma fonte para web de ícones. Permite utilizar vetores escalonáveis que mantem ótima legibilidade em diferentes tamanhos e resoluções de tela. Também permite a formatação de cor, sombras, rotações ou qualquer estilo que possa ser descrito em uma folha de estilo CSS.

Clique aqui para acessar o site do projeto

OwlCarousel <3

OwlCarousel é um plugin construído em jQuery que permite apresentar listas de itens em forma de carrossel. Tem suporte ao design responsivo do portal e também é compatível com interações de toque de dispositivos móveis.
O Portal Adventista utiliza a versão v1.3.2

Clique aqui para acessar o site do plugin

Masonry

Masonry é uma biblioteca de JavaScript que permite controlar o posicionamento vertical de elementos dentro de um grid. Ele re-arranja a disposição de elementos de acordo com a melhor otimização possível, evitando a formação de espaços em branco na página.
O Portal Adventista utiliza a versão v3.1.4.

Clique aqui para acessar o site da biblioteca

Modernizr

Moderniz é uma biblioteca de Javascipt que detecta quais recursos de HTML5 e CSS3 são suportados pelo navegador do usuário. Permite criar condicionais em JavaScript e CSS que permitam adaptar os estilos para cada plataforma de uso.

Clique aqui para baixar a versão compilada para o portal

Clique aqui para acessar o site do plugin

HTML5 Shiv

HTML5 Shiv permite que versões antigas do Internet Explorer (7,8) renderizem elementos de HTML5 de forma correta.

Clique aqui para acessar o site do projeto
