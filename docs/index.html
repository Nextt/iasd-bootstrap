Abas do componente de navegação global

Guia para adicionar ítens/abas ao menu "Mais…" da navegação global.

  1. Criar uma nova location para a aba "Manage Locations" do painel de administração da instalação de taxonomias, no arquivo PANavMaster.class.php, no plugin pa-plugin-taxonomias-master, dentro da função "Init". Manter o padrão de nomenclatura, "multisite_abaX", "Global NX Aba X".
    Ex. : register_nav_menu('multisite_aba3', __('Global N3 Aba 3 (Serviços)', 'iasd'));

  2. Adicionar a location ao array na função "RenderMenus".
    Ex. : $menus['multisite_aba3'] = self::RenderJson('multisite_aba3');

  3. Criar na aba "Aparência", no sub ítem "Menus", no painel de administração da instalação de taxonomias um novo menu.

  4. Assignar o menu criado no ítem anterior na location criada no primeiro ítem.
