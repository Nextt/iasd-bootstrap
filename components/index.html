Recomendamos a utilização da fonte de ícones Font Awesome, composta por mais de 350 ícones vetoriais escalonáveis. No link abaixo você encontra a galeria completa de ícones e instuções de como utilizá-los.

Ver documentação da Font Awesome

Botões

Você pode utilizar as classes padrão do Bootstrap para definir rapidamente tamanho e tipo de botões a serem utilizados.

O estilo vazado foi criado especialmente para o IASD Bootstrap. Para utilizá-lo deve-se adicionar a classe .btn-outline ao <button>. Recomendamos utilizar este padrão apenas para fundos escuros ou com bom contraste. Abaixo um exemplo de código para este estilo de botão.

<div class="styleguide-bg-dark-btn-outline-example">
  <button type="button" class="btn btn-lg btn-outline">Vazado</button>
  <button type="button" class="btn btn-outline">Vazado</button>
  <button type="button" class="btn btn-sm btn-outline">Vazado</button>
  <button type="button" class="btn btn-xs btn-outline">Vazado</button>
  <button type="button" class="btn btn-xs btn-link">Link</button>
</div>

Cabeçalhos

Componente de identificação principal dos sites do Portal Adventista. A versão do IASD Bootstrap oferece 6 variações que atendem diferentes demandas de identidade visual online (listadas a seguir), mas todas compartilham o mesmo código base descrito abaixo.

<header>
  <div class="container">
    <div class="row">
    ...
    </div>
  </div>
</header>
Atenção! Para que os estilo de CSS sejam aplicados corretamente, o elemento <header> deve ser filho direto do elemento <body>.

1. Cabeçalho Institucional

Versão principal do cabeçalho de identificação dos sites do Portal Adventista. Requer que a classe .institutional seja adicionada ao elemento .identifier.

<div class="col-md-7 identifier institutional">
  <div class="brand">
    <a href="#" title="Clique aqui para retornar à página inicial">Igreja Adventista do Sétimo Dia</a>
  </div>
  <div class="title">
    <hgroup>
      <h1>Igreja Adventista do Sétimo Dia</h1>
    </hgroup>
  </div>
</div>

2. Cabeçalho Institucional de Sede Regional

Variação do cabeçalho anterior com a identificação da sede regional (recomenda-se o uso nas instalações de portais de uniões e associações). Requer que as classes .institutional e .headquarters sejam adicionadas ao elemento .identifier.

<div class="col-md-7 identifier institutional headquarters">
  <div class="brand">
    <a href="#" title="Clique aqui para retornar à página inicial">Igreja Adventista do Sétimo Dia</a>
  </div>
  <div class="title">
    <hgroup>
      <h1>Igreja Adventista do Sétimo Dia</h1>
      <h2>Nome da Sede Regional</h2>
    </hgroup>
  </div>
</div>
